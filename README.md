# 📊 Power BI Sales Dashboard (POS Data)

## 📌 Overview
This project presents an interactive **Power BI dashboard** built using point-of-sale (POS) data to analyze sales performance, customer demand, and product trends.

The dataset spans **September 2025 – March 2026 (6 months)** and includes:
- **$495.94K total revenue**
- **61K items sold**
- **$8.08 average price**

## 📊 Dashboard Features

The dashboard includes three key performance indicators (KPIs) to provide a high-level overview of business performance:

- **Total Revenue**
- **Total Items Sold**
- **Average Price per Item**

It also includes interactive visualizations for deeper analysis:

- **Monthly Sales Trend** to identify seasonality and revenue patterns over time  
- **Sales by Day of the Week** to highlight peak customer activity  
- **Revenue by Category** to evaluate product category performance  
- **Top 5 Best-Selling Items** to identify high-demand products  

## 🔍 Key Insights

- **Saturday generated the highest sales**, indicating strong weekend demand  
- **Traditional Milk Tea and Banh Mi** were the top revenue-driving categories  
- **Brown Sugar Boba** was the best-selling product overall  
- Sales experienced a **decline in December** followed by steady recovery into early 2026

The dataset used in this project is not included to protect business confidentiality.

This dashboard enables data-driven decision-making by helping businesses:
